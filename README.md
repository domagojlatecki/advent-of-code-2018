# Advent of Code 2018

Advent of code 2018 task solutions.

### Run
Day 1 - task 1: `cat data/Day1.txt | xargs clj src/Day1-1.clj`  
Day 1 - task 2: `cat data/Day1.txt | xargs clj src/Day1-2.clj`  
Day 2 - task 1: `nim c -r src/Day2Task1.nim < data/Day2.txt`  

### Versions
clj 1.9.0  
nim 0.19.0
