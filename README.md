# Advent of Code 2018

Advent of code 2018 task solutions.

### Run
Day 1 - Task 1: `cat data/Day1.txt | xargs clj src/Day1Task1.clj`  
Day 1 - Task 2: `cat data/Day1.txt | xargs clj src/Day1Task2.clj`
